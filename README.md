## Nội dung

Bài thực hành gồm 5 bài tập:

1. Rút gọn phân số
2. Tìm phân số lớn nhất trong hai phân số
3. Tính tổng, hiệu, tích, thương của hai phân số
4. Tìm ngày kế tiếp
5. Tính điểm trung bình của học sinh

Trong đó bài 1,2,3 được gộp chung vào một file
